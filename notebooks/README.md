# Notebooks

This directory is the executable learning path for the project.

Start Jupyter from the repository root with `uv run jupyter lab`, then follow
the numbered folders and notebooks in order. Keep explanations beside the code
that proves them, and keep tensor shapes visible while a concept is new.

## Conventions

- Use small examples before scaling up.
- Seed random operations when exact values matter.
- Print or assert important tensor shapes.
- Load shared text from `data/`; do not duplicate it in a notebook folder.
- Leave a notebook executable from top to bottom before moving on.
